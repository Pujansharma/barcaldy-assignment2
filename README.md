# barcaldy-assignment2

barcaldy Editor App is a web-based application that allows users to create and customize canvas designs with text, images, frames, and more.


## Features

- **barcaldy Editing:** Add, move, resize, and delete text, images, and frames on the canvas.
- **Text Customization:** Customize text with various font styles, sizes, colors, and alignments.
- **Image Upload:** Upload images from local storage to add to the canvas design.
- **Frame Selection:** Choose from a variety of frame options to enhance the canvas layout.
- **Export as Image:** Export the canvas design as a downloadable image file.
- **Responsive Design:** Works seamlessly across desktop and mobile devices.
